Bienvenido a BoutScout

La aplicación está alojada en GitHub Pages como un sitio web normal, pero el modelo se ejecuta dentro de tu navegador utilizando ONNX Runtime Web. Al abrir esta URL y subir tus archivos CSV, tu navegador carga el modelo y ejecuta la inferencia localmente, permitiendo la revisión manual y la exportación de resultados. Ningún archivo CSV o dato es enviado a ningún servidor externo. ¡Privacidad 100% garantizada!

Este es un proyecto sin fines de lucro, creado desde Latinoamérica para el mundo, con el objetivo de democratizar el análisis del comportamiento aviar.

If you would like to share your post-edition results, discuss incubation patterns, or explore possible collaboration, please feel free to contact us. We are open to collaborative work whenever time and project scope allow.
